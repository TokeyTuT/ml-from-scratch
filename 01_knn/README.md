# 01 KNN 实现指引

## 目标

从零实现 K-Nearest Neighbors 分类器，理解“基于距离查找邻居，再投票”的基本思想。

## 建议文件

- `knn.py`：核心模型类。
- `demo.py`：构造一个小数据集，训练并预测。
- `test_knn.py`：验证距离计算、投票结果、异常输入。

## 实现步骤

1. 保存训练数据 `X_train` 和标签 `y_train`。
2. KNN 是一种**懒惰学习**算法，`fit` 阶段只需要保存训练数据，不需要更新模型参数。
3. 距离函数位于 `util.py`，目前包含欧氏距离、切比雪夫距离和曼哈顿距离。
4. 如果用户不传入距离函数，默认使用欧式距离。
5. 对每个待预测样本，计算它到所有训练样本的距离。
6. 找出距离最近的 `k` 个样本。
7. 对这 `k` 个样本的标签做多数投票。
8. 返回预测标签。

## 注意点

- `k` 必须大于 0。
- `k` 不应大于训练样本数量。
- 输入可以先只支持二维特征矩阵。
- 如果投票平票，可以先选择距离更近的类别，或者选择排序后第一个类别。

## 最小验收

- 能在二维玩具数据上正确分类。
- `fit` 后再调用 `predict`。
- `predict` 支持一次预测多个样本。
